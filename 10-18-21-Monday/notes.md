## Boilerplate code

We'll be using Vite instead of `create-react-app`. The folder structure may differ from the slides.

To get react boilerplate code, run

```
npm init vite@latest my-app -- --template react
```

Now to start your project, run

```
cd my-app
npm install
npm run dev
```
